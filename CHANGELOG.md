# 更新日志

## v2.7.7

### 新功能

- **每周番表与订阅体系** — 新增日历视图、外部番订阅、批量订阅与订阅轮询能力。
- **Trakt 元数据源接入** — 新增 Trakt OAuth、日历、搜索、详情与用户观看状态同步。
- **批量管理日历模式** — 批量管理新增列表/日历双视图，支持外部来源筛选、搜索与批量操作。

### 修复

- **日历订阅状态问题** — 修复订阅状态刷新、取消订阅、本地追更误影响及用户授权上下文相关问题。
- **修复自动刮削任务** — 修复 romaji字段的识别与写入判断。
- **批量订阅交互问题** — 修复批量确认、移动端批量操作、全选/取消全选状态显示异常。

### 优化

- **前端 i18n 改造** — 完成前端多语言抽取与文案统一，补齐简体中文、繁体中文与英文语言包。
- **界面UI优化** — 部分界面样式优化
- **元数据与任务体验优化** — 优化元数据源、通知渠道、任务列表、Webhook、备份与维护任务的状态展示。
- **主页搜索分页优化** — 优化缓存写入、缓存数据读取、提高接口响应速度。
- **重构交互模块** — 优化交互逻辑。

### 维护

- **外部日历数据结构扩展** — 扩展外部订阅、本地强关联、跨源 ID 与平台用户状态字段。
- **订阅与追更任务重构** — 优化订阅轮询、自动导入、缺集补全、自动完结与日程同步协作流程。
